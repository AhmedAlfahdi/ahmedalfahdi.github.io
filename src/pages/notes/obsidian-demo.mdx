## Code Blocks

JavaScript example:

```javascript
function fibonacci(n) {
  if (n <= 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);
}

console.log(fibonacci(10)); // 55
```

Python example:

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

## Tips for Obsidian Users

1. **Export your notes**: Copy `.md` files from your Obsidian vault
2. **Add frontmatter**: Include title, description, and layout
3. **Convert wiki-links**: Change `[[Note]]` to `[Note](/notes/note)`
4. **Mermaid diagrams**: Use the `<Mermaid>` component for diagrams
5. **Math**: LaTeX math works automatically with `$` and `$$`
6. **Images**: Store in `/public` folder and reference with `/image.png`
