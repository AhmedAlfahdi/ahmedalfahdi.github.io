# Embedding 3D Models in Markdown

This page demonstrates how to embed your own 3D models in markdown files.

## Basic Usage

Simply use the `<Model3D>` component with the path to your model file:

```mdx
<Model3D src="/models/your-model.glb" />
```

## Custom Configuration

### Adjust Size and Position

```mdx
<Model3D 
  src="/models/your-model.glb" 
  height="600px"
  scale={2}
  position={[0, -1, 0]}
/>
```

### Control Rotation

```mdx
<Model3D 
  src="/models/your-model.glb"
  rotation={[0, Math.PI / 4, 0]}
  autoRotate={false}
/>
```

### Custom Background

```mdx
<!-- Transparent background -->
<Model3D src="/models/your-model.glb" background="transparent" />

<!-- Custom color -->
<Model3D src="/models/your-model.glb" background="#ff6b6b" />
```

## Component Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `src` | string | required | Path to model file |
| `height` | string | `"500px"` | Container height |
| `scale` | number | `1` | Model scale multiplier |
| `position` | [x,y,z] | `[0,0,0]` | Model position offset |
| `rotation` | [x,y,z] | `[0,0,0]` | Initial rotation (radians) |
| `autoRotate` | boolean | `true` | Auto-rotate model |
| `background` | string | `"default"` | Background color or "transparent" |

## Supported Formats

**Recommended: GLTF/GLB**
- Includes textures, materials, animations
- Best compression
- Industry standard

**Also Supported: OBJ**
- Basic geometry only
- No textures/materials
- Simpler format

## Finding 3D Models

Download free models from:
- [Sketchfab](https://sketchfab.com/) - Largest collection
- [Poly Pizza](https://poly.pizza/) - Low-poly models
- [Quaternius](http://quaternius.com/) - Free game assets
- [KenShape](https://kenney.nl/assets) - Free game assets

## Example: Multiple Models

You can embed multiple models on one page:

```mdx
<Model3D src="/models/model1.glb" height="400px" />
<Model3D src="/models/model2.glb" height="400px" />
```
