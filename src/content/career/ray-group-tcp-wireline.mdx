Six-month OPAL (EMDAAD) program covering wireline logging and TCP operations with hands-on explosive-handling compliance.

## Highlights
- Supported open- and cased-hole logging (SP, Sonic, Resistivity, Density, Neutron, CBL, MIT).
- Assembled TCP strings (gun systems, firing heads, packers/BPVs) following HES procedures for HMX, RDX, PENT.
- Assisted fiber-optic perforation setups and pre-job readiness checks/documentation.
- Coordinated readiness checks, tool string prep, and onsite QA for field deployments.

## Tools & Skills
- Wireline / TCP field operations, explosive handling, fiber-optic perforation prep.
- Safety-first execution; documentation and pre-job checklists.
