# 3D Models Directory

Place your 3D model files (.gltf, .glb, .obj) in this directory.

## Supported Formats

- **.gltf / .glb** - Recommended format (includes textures, animations)
- **.obj** - Basic geometry only

## Usage

Reference models from your markdown files:

```mdx
<Model3D src="/models/your-model.glb" />
```

## Example Models

You can download free models from:
- [Sketchfab](https://sketchfab.com/feed)
- [Poly Pizza](https://poly.pizza/)
- [Quaternius](http://quaternius.com/)
- [KenShape](https://kenney.nl/assets)

## File Size Tips

- Keep models under 5MB for good performance
- Use GLB format for better compression
- Optimize textures (max 2048x2048)
- Reduce polygon count when possible
